Web Develpment

Web development is the work involved in developing a website. This includes building, creating and maintaining the web page. Web development can range from developing a simple static page of plain text to complete web applications. Everything that you see on the website is built with the help of front-end development (also known as front-end web development) and the people making it happen are known as Front end developers. Web developers are the fundamental part of any tech team or can work as freelancers. Different technologies are used by web developers to build websites which will be discussed below.

FrameWorks And Libraries

FrameWorks:

As a developer, you don’t need to start from scratch when tools are designed to help you. Frameworks are developed and used by developers to build web apps. The aim of frameworks is to provide a common structure so the developers don’t have to redo from scratch.

Cascading Style Sheet (CSS) Frame Works:

  • Bootstrap
  • Tailwind CSS
  • Bulma
  • Skeleton


JavaScript Frame Works:
  • Angular
  • Vue.js
  • Ember
  • Express.js

Libraries:

A library is a collection patterns or base foundation to write code while it provides structure and quick ways to solve problems. These implement different behavior and action on your web page.

Following are the JavaScript libraries:

  • jQuery
  • React
  • Underscore.js
  • D3.js

    The technical difference between a framework and library lies in a term called inversion of control. When you use a library, you are in charge of the flow of the application. … When you use a framework, the framework is in charge of the flow.
